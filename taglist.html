Content Tags

Heading Levels

Heading levels are titles for sections of information. There are 6 and next within each other where heading level 1 is the highest.

<h1> ... </h1> is just one example.

Paragraph

Used to hold text or information in its simplest form. inline tags can be used within to enhance experience.

<p> ... </p>

Lists

Lists can be ordered and unordered. ordered lists will be labeled numerically, while unordered lists will start with bulletpoints.

Unordered List Example

HTML Code

<ul>
  <li>item in list</li>
  <li>another item in the list</li>
</ul>

Output

  • item in list
  • another item in the list

Ordered List Example

HTML Code

<ol>
  <li>item in list</li>
  <li>another item in the list</li>
</ol>

Output

  1. item in list
  2. another item in list
