HTML Document Setup

File Type: .html

First Line Declaration: <!doctype html>

Encapsulate all html elements in between the <html> ... </html> tags.

The Head

Encapsulate your document information in <head> ... </head>

Basic tags used in Head

  • <title> ... </title>
      - Used to hold the title of the document.

  • <link rel="stylesheet" type="text/css" href="mystyle.css">
      - Used to link your CSS stylesheet.

The Body

The body tag goes below the head and all of your content will go within your body.

Use the tag <body> ... </body>

Use Semantic HTML to better define your layout

Semantic provides structure to the document. All of these tags represent sections of your file and makes it easier to understand and read. In the list below they will be presented in the order of use.

  1. <header> ... </header>
      - Used to contain heading information like the title, author, date, etc...

  2. <main> ... </main>
      - Used to contain the dominant content of the page.

  3. <section> ... </section>
      - Used to define sections of the document. You can put articles and other information within them.

  4. <article> ... </article>
      - Used to section hold self contained content. All of the information is perhaps about one subject. this tag will be used within your <main> tag.

  5. <aside> ... </aside>
      - Holds content within <article> that is indirectly related to the content.

These were just a simple set of tags for semantic HTML. Click this link to go to W3schools list of semantic tags and their definitions.
